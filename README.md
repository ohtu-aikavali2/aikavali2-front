# Definition of Done

- The feature has to be planned and documented accordingly.
- The feature has to pass testing on the Travis CI server.
- The feature has been approved by the customer.

## Start  
Install dependencies:

```npm install```

Start the project:

```npm start```

## Additional scripts
Tests:

```npm test```

Linter:

```npm run lint```
